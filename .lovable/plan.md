

# Политика конфиденциальности для приложения STSphera CITY4

## Что будет сделано

Добавление страницы "Политика конфиденциальности" в соответствии с требованиями законодательства РФ (ФЗ-152 "О персональных данных" в актуальной редакции 2026 года).

## Состав документа

Политика будет включать все обязательные разделы по ФЗ-152:

1. **Общие положения** -- наименование оператора (STSphera), цели обработки ПДн
2. **Правовые основания** -- ФЗ-152, ТК РФ, ГК РФ, согласие субъекта
3. **Перечень обрабатываемых данных** -- ФИО, должность, контактные данные, PIN-код, роль в системе
4. **Цели обработки** -- управление проектами, идентификация пользователей, учёт персонала
5. **Порядок сбора и хранения** -- автоматизированная обработка, сроки хранения
6. **Права субъекта ПДн** -- доступ, изменение, удаление, отзыв согласия
7. **Защита персональных данных** -- технические и организационные меры
8. **Трансграничная передача** -- указание что данные хранятся на территории РФ (или указание серверов)
9. **Контактная информация** -- ответственный за обработку ПДн, адрес для обращений
10. **Заключительные положения** -- дата вступления в силу, порядок изменений

## Технические изменения

- Полный текст политики конфиденциальности на русском языке
- Оформление в стиле приложения (тёмная тема, шрифты)
- Кнопка "Назад" для возврата
- Дата: февраль 2026

### 2. Маршрут в `src/App.tsx`
- Добавить роут `/privacy` для страницы политики

### 3. Ссылка на экране входа `src/components/LoginScreen.tsx`
- Внизу экрана логина добавить ссылку "Политика конфиденциальности"
- Ссылка ведёт на `/privacy`

## Технические детали

- Страница доступна без авторизации (не требует PIN)
- Используется `react-router-dom` Link для навигации
- Текст политики встроен в компонент (не требует БД)
- ScrollArea для удобной прокрутки длинного документа
